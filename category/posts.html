Deep learning model to predict the stock prices of a company¶

The dataset used here is a set of market closing prices of S&P; 500 (INDEXSP: .INX) and can be found here : https://www.kaggle.com/mdquigg/sp500¶

There are two files in this project. The first is a wrapper for the LSTM class utilizing Keras with Tensorflow as backend, while the second is implementing a time series LSTM. More information about LSTM can be found out in this link : http://colah.github.io/posts/2015-08-Understanding-LSTMs/
